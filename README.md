# BeautyybyyKiKi.com ✨
A premium, high-fashion website for a professional black-owned hair styling brand.

## Features
- **Luxury Aesthetic**: Matte gold and deep black design system.
- **Service Showcase**: High-resolution imagery for Braids, Sew-ins, and Custom Wigs.
- **Glassmorphism**: Modern UI effects for buttons and cards.
- **Responsive**: Seamless experience on all devices.
- **Scroll Animations**: Dynamic reveal effects for content.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Playfair Display, Inter, Outfit)

## Deployment
This website is ready for deployment on Netlify, Vercel, or GitHub Pages.
